# Quick Poll - Decision Maker Frontend

. Create polls with custom questions and options
. Vote on active polls
. See beautiful real-time results with percentages
. Track poll expiry times
. View statistics on the dashboard
. Filter polls (active, expired, all)

# Tech Stack 
. React 19 - Latest React with improved hooks and features
. Vite - Super fast build tool (way better than Create React App imo)
. Tailwind CSS 4 - For styling (those gradients are chef's kiss)
. React Router v7 - For page navigation
. Axios - For API calls
. Framer Motion - For smooth animations
. ESLint - To catch our mistakes before they happen
